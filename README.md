# Solar Circuits
Additional resources and a collection of programs exploring solar powered computing.

All programs have been tested with ESP32 C3 microcontrollers.

## Projects
### Projects in the Book
1) Hello Sun - A simple 'Hello World' program for reading supercapacitor status and flashing the LED.

### Additional Solar Computing Projects
2) Solar Simon - A simple game using the solar power as a game mechanic.
3) Solar Hotspot - A simple solar powered wifi network
4) E-ink Project
5) INA219

## Installation Instructions
1) Download the Arduino IDE onto your computer from the Arduino website. https://www.arduino.cc
2) Open the IDE and navigate to the Board's Manager. You will find the board’s manager on the menu on the left side of the window. Search for ‘esp32 Espressif Systems’ and click install.
3) Connect your board to your computer with a USB cable.
4) Once connected, from the drop down menu on top, click ‘Select other board and port…’
5) Search for the Esp32 C3 board. A large number of different types of Esp32 C3 boards will show up. Select ‘ESP32C3 Dev Module’ unless you are using a specific board.
6) In the ‘Tools’ drop down menu, enable ‘USB CDC On Boot’
